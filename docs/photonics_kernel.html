class PhotonicsKernel(builtins.object)
    PhotonicsKernel(function)
 
The `PhotonicsKernel` class serves as to process CUDA-Q kernels for the `photonics` target.
The target must be set to `photonics` prior to defining a `PhotonicsKernel`.
 
The quantum operations in this kernel apply to qudits defined by `qudit(level=N)` or a list of qudits.
The qudits within a kernel must be of the same level.
 
Allowed quantum operations are:  `plus`, `phase_shift`, `beam_splitter`, and `mz`.
